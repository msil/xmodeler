Basic Types

XOCL provides four basic types

  • Boolean

  • String

  • Integer

  • Float (64 bit real numbers)

All the usual operators, e.g. arithimetic operators, are provided (see the Bluebook for a detailed list).

XOCL also provides the inbuilt value "null" for situations when the result of an expression cannot be evaluated.

Examples

Type the following into the console:

        1 + 2;
        10.mod(3);
        true and false;
        true implies false;
        "Hello" + "World";
        "Hello".toUpper();
        10.1.cos();
        3.14.sqrt();
        null;
