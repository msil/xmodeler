Running Mappings

Mappings are run by instantiating the mapping and passing it the appropriate object/s to be mapped. Thes syntax of a call is as follows:

<Mapping>()(<Value>)

Note that if there is more than one domain, this must be reflected by the list of values passed to the mapping, e.g.

<Mapping>()(<Value1>,<Value2>,..,ValueN)
