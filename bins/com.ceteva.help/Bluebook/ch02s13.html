Adding Getters and Setters

Getter and Setter operations for specific attributes can be conveniently added to a class. Right click on the attribute (in the diagram or browser) and select Create Getter or Creat Setter.

Creating a Setter will add operations for adding new instances to the attribute (as shown below), while adding a Getter will add an operation to query the value of an attribute.
